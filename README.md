# EvoAgent

## What is EvoAgent?

Imagine if Darwin's theory of evolution could be applied to AI agents in real-time. That's EvoAgent.

**For newcomers**: EvoAgent is like a digital ecosystem where AI agents compete, learn, and evolve automatically‚Äîjust like species in nature. The weak agents die out, the strong ones survive and reproduce, and over time you get incredibly capable AI that adapts to any challenge.

**For technical users**: EvoAgent is an evolutionary multi-agent framework built on AS-FDVM (Adaptive Search - Fractal Darwinian Virtual Machines) architecture. It implements competitive evolution, mutation-based adaptation, and fitness-driven selection to create self-improving agent populations.

## How It Works: Evolution in Action

### The Core Mechanism

1. **Agent Population**: Start with multiple AI agents (like different "species")
2. **Competitive Tasks**: Agents compete on real challenges and problems  
3. **Natural Selection**: Best-performing agents survive and "reproduce"
4. **Mutation & Evolution**: Offspring agents inherit traits with random variations
5. **Continuous Adaptation**: The cycle repeats, constantly improving capabilities

### Key Components

- **Evolving Graph Memory**: Each agent maintains a living knowledge graph that grows smarter over time‚Äînot just remembering facts, but building interconnected understanding
- **Competitive Agent Mixing**: Multiple agents don't just compete‚Äîthey also collaborate when it benefits survival, creating emergent team strategies
- **Lineage Tracking**: Full genealogy for every agent‚Äîtrace any breakthrough back through its evolutionary ancestry to understand how it emerged
- **Cross-Domain Adaptation**: Agents that excel in one domain automatically apply learned strategies to completely different problems

## Getting Started

### For Users: Quick Start

1. **Clone and Setup**
   ```bash
   git clone https://github.com/ftshortt/evoagent.git
   cd evoagent
   pip install -r requirements.txt
   ```

2. **Choose Your Agent Type**
   ```bash
   export EVO_AGENT=qwen   # or r1, ocr
   ```

3. **Launch Evolution**
   ```bash
   # Single machine
   python -m src.agent_orchestrator
   
   # Multi-GPU
   torchrun --nproc_per_node=NUM_GPUS -m src.agent_orchestrator
   ```

4. **Watch the Magic**: Agents compete, evolve, and improve automatically
